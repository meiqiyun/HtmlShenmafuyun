昨天的 Model 3 交付仪式后，越来越多的人把特斯拉比作苹果，并认为 Model 3 堪比苹果标志性产物 iPhone 4。

从昨天的交付仪式来看，至少有一点，特斯拉与苹果有明显的不同——公布会。

特斯拉的发布会，更像是一场 Elon Musk 的小我私家聚会，或者说，属于 Elon Musk 的表演时候。

乔布斯辞世后的苹果发布会，本质上已经变成一家大型公司举办的有等级、有秩序、有仪式感的发布会。

参加特斯拉发布会的人，更像是听了 Elon Musk 的一场脱口秀。而对 Elon Musk 的崇拜，正是很多人购买特斯拉电动车的重要原因。

如此巨大的销量不仅仅意味着更多的电动车市场份额，还能在2年多的时间内给特斯拉带来上百万的 Autopilo 用户和上百万辆天天在道路上收集主动驾驶技术数据的终端。这会帮助特斯拉在自动驾驶技术上实现快速的成长。

Elon Musk 在发布会上如许评价 Model 3：「今天是特斯拉的大日子，Model 3 是款巨大且代价亲民的电动车。世界上有伟大的汽油车，有伟大的豪华车，但是没有过伟大且价格亲民的电动车。」

但硬币都有两面。Model 3 的上市，对特斯拉来说也会有负面影响。

之前乐于花近百万元人民币购买 Model S 的用户，大多数看中的都是特斯拉电动车极致的加快性和 Autopilot 自动驾驶帮助功能。讲真，现在 S/X 的工艺质量还达不到百万元豪车的尺度。

而这两者，Model 3 都能提供。

零百公里加速参数上，Model 3 会比 Model S 慢零点几秒，但对绝大多数用户而言，体验不出太大的差别。凭据这次 Model 3 交付仪式前的试驾反馈，有很多媒体都对 Model 3 的加速性大加称赞。

而 Model 3 上配备了和 Model S 相同的 Autopilot 硬件配置，只需选购了 Autopilot 服务就能启用这个高级自动驾驶辅助系统。

从这两方面来看，Model 3 就是一款价格更便宜、空间更小的 Model S。那购买者为什么要花7万美元而不是3.5万美元买一辆特斯拉电动车？
